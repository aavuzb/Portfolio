Multi-Scale Template Matching.2021, 2023

The goal of this research is to control multiple devices at the same time. To implement this task, I used “Multi-scale template matching” open-source code (link: https://github.com/agiledots/multiscale-template-matching), However, I have made new formulas that help to decrease the matching time and increase the matching result. This research has 2 versions. 1st version was implemented in 2021 and 2nd version was implemented in 2023. The difference is 1st the version supports small-sized template image, maximum template image size must not be greater than 40x40 in 1st version (Video 1), however, 2nd version supports all kinds of sizes (Video 2). The patent was published for 1st version of MSTM. And, another patent will be written for 2nd version soon.

  • Features:
    • Template matching.
    • Feature matching
    • Threading
    • Rest API
    • etc.

  • Platform: Ubuntu, Windows, MacOS
  • Stack: Python, C++, OpenCV, Multi-Scale Template Matching
