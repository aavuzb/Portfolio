Video 1. Café Cleanlines

Café Cleanlines.2019-2020

This project was related to Object Recognition. The goal is to notify a restaurant or cafe employee about whether the tables are empty or not. That Recognition model has 4 classes:
1) Empty + Unoccupied: The table is empty and there is no customer behind that table;
2) Not_Empty + Unoccupied: There is something on a table and there is no customer behind that table
2) Empty + Occupied: The table is empty and there is a customer behind that table;
4) Unoccupied + Occupied: here is something on a table and there is no customer behind that table

Several pre-trained classification models are used to make a recognition model. However, Efficient Net B3 and Efficient Net B6 pre-trained models gave much better classification results than Inception, Mobile Net, and Efficient Nets [B0, B1, B2, B4, B5, B7]

  • Features:
    • Object Classification&Recognition
    • Transfer Learning
    • Efficient Nets
    • etc.

  • Platform: Ubuntu, Windows
  • Stack: Tensorflow, OpenCV, Efficient Nets
