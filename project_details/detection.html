Object Detection Models & APIs

Three kinds of Object Detection models were made using YoloV5 and their Rest APIs were implemented by Flask. 1st one is used to detect all objects on Mobile Applications, its name is a Multiple-object detector. Classes are Icon, Text, Picture, etc (Video 1). The next one is used to find Search Icons or Search Buttons on Text-entering pages of Mobile applications (Video 2). The last one is utilized to detect or classify Loading objects and Loading pages (Video 3). Rest APIs were implemented for all detection models shown in Videos 1-3.

  • Features:
    • Data Collection
    • Data Labeling
    • Data Augmentation
    • Transfer Learning
    • Rest API
    • etc.

  • Platform: Ubuntu, Windows, MacOS
  • Stack: PyTorch, Tensorflow, OpenCV, Flask, Yolo
